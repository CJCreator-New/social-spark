# Thematic — Code Duplication

- **`dashboard-deploy/*.ts`** (`decrypt-api-key.ts`, `encrypt-api-key.ts`, `generate-calendar.ts`, `generate-single-post.ts`, `generate-post-image.ts`, `regenerate-post.ts`, `verify-payment.ts`, `create-order.ts`, `mcp.ts`, `cleanup-media.ts`, `delete-api-key.ts`, `health.ts`, `inline-rewrite.ts`, `queue-worker.ts`, `repurpose-post.ts`, `generate-trends.ts`, `trends_read.ts`, `telemetry.ts`) mirror `supabase/functions/*/index.ts`. Presumably a deploy bundle. Divergence risk if only one is edited (git-diff between the two is a good regression check).
- **CORS + JSON-response boilerplate** duplicated in each edge fn (`create-order:11-22`, `verify-payment:38-49`, `encrypt-api-key:19-30`, `delete-api-key:7-18`, etc.). Shared helper `jsonResponse` exists (`_shared/promptHelpers.ts` — used by AI functions) but the auth/payment functions each declare their own local `jsonResponse`. Standardize.
- **Admin identity checks** in RLS: pattern `EXISTS(SELECT 1 FROM user_roles WHERE user_id=auth.uid() AND role='admin') OR EXISTS(SELECT 1 FROM admin_users WHERE user_id=auth.uid())` appears in `20260609124000_api_key_security_hardening.sql:37,116-129`, `20260616000000_subscription_tiers.sql:44`, `20260617000000_admin_comp_grants.sql:16`. `is_admin()` helper exists and should replace inline copies.
- **`AdminRoute` and `ProtectedRoute`** share identical 8-second timeout scaffolding — could hoist into a shared hook `useLoadingTimeout(ms)`.
- **Route `<Suspense fallback={<RouteFallback title="…"/>}>` boilerplate** repeated 12× in `App.tsx:66-186`. A small `<LazyRoute title=…>` wrapper would cut this in half.
